# archived Docker Compose files

這裡的 Docker Compose files 只作 historical reference。

它們不支援、不維護為 installation sources，也不屬於 main branch canonical install flow。它們可能已經過期，也可能無法配合目前 Docker image、runtime paths、permission model 或文件內容運作。

正式支援的 install source：

- `../../docker-compose.yml` — root 的 single-container canonical Docker Compose file

已歸檔的歷史檔案：

- `docker-compose.two-container.yml`
- `docker-compose.three-container.yml`
