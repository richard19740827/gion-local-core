# archive

這個目錄只作 historical reference。

archive 內的檔案：

- **not supported**
- **not tested as an install source**
- **不屬於 main branch canonical install flow**
- **不保證能 build、start，或與目前 runtime 行為相容**

正式支援的 Docker install source 只有 root 的 `docker-compose.yml`。
`archive/docker/` 內的 Docker Compose files 只用來保留舊 multi-container 實驗與 migration 歷史，方便日後回溯，不作安裝來源。
